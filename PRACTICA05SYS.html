INTRODUCCION

La integración númerica es una herramienta que ayuda a aproximar numéricamente los valores de integrales definidas, las cuales son dificiles de calcular o no se pueden solucionar por metodos analíticos. Para ello, existen metodos que ayudan a resolver este tipo de integrales y se les conoce cómo métodos de integración numérica. A continuación se pueden observar algunos ejemplos.

PROBLEMA 1

u= @(t) 1.0.*(t>=0);
tri = @(t) (1-abs(t)).*u(t/2);
pul = @(t) u(t)-u(t-1);
 
f= @(t) (tri(t).*pul(t))+(tri(-t+2).*pul(t-1));
h= @(t) pul(t);
I= @(t) pul(-t);
g= @(t) I(t).*f(t);
 
dt=1e-3;t= -3:dt:3;
 
subplot(4,1,1);plot(t,f(t),"b");grid on;
subplot(4,1,2);plot(t,h(t),"b");grid on;
subplot(4,1,3);plot(t,I(t),"b");grid on;
subplot(4,1,4);plot(t,g(t),"b");grid on;

PROBLEMA 2

u= @(t) 1.0.*(t>=0);
tri = @(t) (1-abs(t)).*u(t/2);
pul = @(t) u(t)-u(t-1);
 
f= @(t) (tri(-t+1).*pul(t))+pul(t-1);
h= @(t) u(t-1)-u(t-3);
I= @(t) u(-t-1)-u(-t-3);
g= @(t) I(t).*f(t);
 
dt=1e-3;t=-4:dt:4;
 
subplot(4,1,1);plot(t,f(t),"b");grid on;
subplot(4,1,2);plot(t,h(t),"b");grid on;
subplot(4,1,3);plot(t,I(t),"b");grid on;
subplot(4,1,4);plot(t,g(t),"b");grid on;
